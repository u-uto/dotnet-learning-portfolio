# Type（型）
型とは：データの種類を表すもの

## 概要
データの種類を定義する仕組み

---

## 主な型

- int：整数
- double：浮動小数
- string：文字列
- bool：真偽値
- DateTime：日時

---

## 値型と参照型

### 値型
値そのものを保持

### 参照型
参照先を保持

---

## なぜ必要か
扱える値や動作を制限することで、バグを防ぐため

---

## 実務での使用例
- 金額 → int / decimal
- フラグ → bool
- 日付 → DateTime

---

## 一言まとめ
「データのルールを決めるもの」
