# Operator（演算子）
演算子とは：値に対して処理を行う記号

## 概要
値を計算・比較・評価するための記号

---

## 主な種類

### 算術演算子
\+ - * / %

### 比較演算子
== != > < >= <=

### 論理演算子
&& || !

### 代入演算子
= += -=

---

## ポイント
- 優先順位に注意
- 意図が伝わる書き方をする

---

## なぜ必要か
データ同士を組み合わせてロジックを作るため

---

## 実務での使用例
- 金額計算
- 条件分岐
- フラグ判定

---

## 一言まとめ
「値同士をつなぐルール」
