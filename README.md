# dotnet-learning-portfolio
# .NET Learning Portfolio

## 概要
C# / .NETの基礎から設計・Webアプリケーションまでを体系的に整理したリポジトリ

---

## 学習方針
- コードだけでなく設計意図（Why）を重視
- Before / Afterで改善を理解
- 実務での利用を前提に整理

---

## 構成

### 01_Fundamentals
言語基礎（型・制御構文など）

### 02_OOP
オブジェクト指向（設計の基礎）

### 03_DataHandling
コレクション・データ操作

### 04_ErrorHandling
例外処理・ログ

### 05_AsyncProgramming
非同期・並列処理

### 06_Architecture
設計（DI / Repository / レイヤー構造）

### 07_WebApplication
ASP.NET Coreによる実践

---

## 強み
- 設計理由をREADMEに明文化
- 比較コードによる理解
- 実務適用を意識した構成
